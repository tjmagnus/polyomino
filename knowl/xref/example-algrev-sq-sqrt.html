Example A.4.33. Computing Squares and Square Roots.

Compute the following using the definitions of square and square root. A calculator should not be necessary.
  1. \(\displaystyle 5^2\)
  2. \(\displaystyle \sqrt{400}\)
  3. \(\displaystyle (\sqrt{400})^2\)
  4. \(\displaystyle \sqrt{5^2}\)
Answer.
a) \(25\text{,}\) b) \(20\text{,}\) c) \(400\text{,}\) d) \(5\)
