Task 4.1.2.a.
Plot the images \(A'\text{,}\) \(B'\text{,}\) \(C'\text{,}\) and \(D'\) of the points \(A\text{,}\) \(B\text{,}\) \(C\text{,}\) and \(D\) across the horizontal (\(x\)-) axis. Determine the coordinates of each new point. Create the quadrilateral \(A'B'C'D'\) and color it red. What effect does reflecting across the \(x\)-axis have on coordinates?
If you choose to do this activity online using the GeoGebra applet in Figure 4.1.11, you may enter a point using the Input box on the bottom of the applet. Type “\(A'=(4,10)\)” and GeoGebra will plot the point \((4,10)\) and label it \(A'\text{.}\) Of course, that is not the correct image for \(A'\text{!}\) You might not be able to color code the quadrilaterals.
